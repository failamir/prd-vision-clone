## Data Flow Details

### Candidate Registration Flow
```
Candidate ──[Registration Data]──▶ Process 1.0 ──[User Record]──▶ D1: User Data
                                         │
                                         └──[Profile Data]──▶ D1: candidate_profiles
```

### Job Application Flow
```
Candidate ──[Application]──▶ Process 3.0 ──[App Record]──▶ D4: job_applications
     │                            │
     └──[CV Upload]──▶ Process 4.0 ──[File]──▶ Storage: cvs bucket
```

### Interview Scheduling Flow
```
Admin ──[Schedule Data]──▶ Process 5.0 ──[Update Status]──▶ D4: job_applications
                               │
                               └──[Notification]──▶ Process 7.0 ──▶ D5: messages
```
